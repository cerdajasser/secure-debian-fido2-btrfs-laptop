# Placeholders used in this repo

| Placeholder | Meaning | Get it with |
|---|---|---|
| `<USER>` | Local Linux username | `whoami` |
| `<HOSTNAME>` | Hostname | `hostname` |
| `<UUID>` | Generic filesystem/LUKS UUID | `lsblk -f` or `blkid` |
| `<LUKS_UUID>` | LUKS partition UUID | `lsblk -f` or `sudo blkid /dev/<encrypted-partition>` |
| `<BOOT_FS_UUID>` | `/boot` filesystem UUID | `sudo grub-probe --target=fs_uuid /boot` |
| `<KERNEL_VERSION>` | Kernel version | `uname -r` |
| `<ROOT_MAPPER>` | Root mapper path | `findmnt -no SOURCE /` |
| `<VG_NAME>` | LVM volume group/mapper prefix | `lvs` or `findmnt -no SOURCE /` |
| `<ENCRYPTED_PARTITION>` | LUKS device path | `lsblk -f` |
| `<key-public-mapping>` | U2F mapping data | generated by `pamu2fcfg`, do not publish real value |
