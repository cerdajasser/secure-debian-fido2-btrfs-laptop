# Do not publish these

- Real `/etc/Yubico/u2f_keys`
- LUKS recovery key
- Full `cryptsetup luksDump` output unless you have reviewed every line
- SSH private keys
- Bitwarden/password manager data
- Browser profiles
- Private tokens/API keys
- Full home directory dumps
- Full `/boot/grub/grub.cfg`
- Device serial numbers
